# Travel-planner-AI
This AI take inputs from the frontend which is primarily streamlit and then sends it to Gemini AI for computing . Gemini basically gives the place to stay( eg.Resort , Beach) , gives the weather , the itinerary of the things you can do

The secrets.toml file in the file list has to be updated with your own API key (GEMINI API) . As this works on Gemini 2.5 Pro platform this is the requirement
